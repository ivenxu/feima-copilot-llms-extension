# Global Region Assets

This directory contains region-specific assets for the Global (US/EU) extension.

## Contents

- `icon.png` - Extension icon for VS Code Marketplace (Global region)
- `screenshots/` - Screenshots showcasing the extension features in English

## Icon Requirements

- **Format**: PNG
- **Size**: 128x128 pixels (recommended)
- **Color**: Full color or grayscale
- **Transparency**: Alpha channel supported

## Screenshots

Create 3-5 screenshots showing:
1. Code completion in action
2. English UI localization
3. Model selection interface
4. Authentication flow
5. Settings panel

**Naming Convention**: `screenshot-1.png`, `screenshot-2.png`, etc.
