# CN Region Assets

This directory contains region-specific assets for the China extended.

## Contents

- `icon.png` - Extension icon for VS Code Marketplace (CN region)
- `screenshots/` - Screenshots showcasing the extension features in Chinese context

## Icon Requirements

- **Format**: PNG
- **Size**: 128x128 pixels (recommended)
- **Color**: Full color or grayscale
- **Transparency**: Alpha channel supported

## Screenshots

Create 3-5 screenshots showing:
1. Code completion in action
2. Chinese UI localization
3. Model selection interface
4. Authentication flow
5. Settings panel

**Naming Convention**: `screenshot-1.png`, `screenshot-2.png`, etc.
